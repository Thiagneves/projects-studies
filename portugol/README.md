# Portugol - Exercícios e Prática

Este repositório reúne exercícios e exemplos desenvolvidos em Portugol para o aprendizado de lógica de programação.

## 📚 Conteúdo

- Estruturas de decisão (if/else)
- Estruturas de repetição (for, while)
- Variáveis e tipos de dados
- Entrada e saída de dados
- Resolução de problemas básicos

## 🎯 Objetivo

Desenvolver o raciocínio lógico e compreender os fundamentos da programação de forma simples e estruturada.

## ⚠️ Observação

Este repositório é destinado apenas para fins de estudo e prática.
